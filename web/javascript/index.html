Event Listeners

  • We can tell the browsers when a javaScript function should be exicuted
  • It is done using the event Listeners
  • Common event listeners
    onClick
    Occures when a user clicks an object in webpage
    onMouseEnter
    Occures when mouse pointer is moved onto an object
    onMouseLeave
    Occures when mouse pointer is moved out of an object
    onKeyDown
    Occures when user is pressing a key in the keyboard
    onKeyUp
    Occures when user isreleasing a key in the keyboard
